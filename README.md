Проект представляет из себя Spring Boot Web MVC приложение для управления библиотекой. Система позволяет работать с книгами, читателями и выдачами книг, используя базу данных PostgreSQL и 3 связанные таблицы.

## Функциональность
CRUD для книг: добавление, редактирование, удаление, просмотр.
CRUD для читателей: добавление, редактирование, удаление, просмотр.
CRUD для выдач книг: оформление выдачи, возврат, просмотр.
Валидация данных на уровне сущностей (например, проверка формата ФИО, телефона, email, даты рождения, года публикации).
